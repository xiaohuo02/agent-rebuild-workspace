# 04-测试与评测设计

## 测试分层

第一阶段包含三类测试：

1. Router Test（路由测试）：验证用户问题能分到正确类型。
2. Tool Test（工具测试）：验证工具返回统一结构。
3. Agent Flow Test（智能体流程测试）：验证完整链路能返回 answer、trace 和 evidence。

## DeepEval 评测骨架

第一阶段先不接 DeepEval（大模型评测框架）真实执行器，只准备数据集和指标设计。

后续评测指标：

| 指标 | 中文含义 | 作用 |
|---|---|---|
| Answer Relevancy | 答案相关性 | 回答是否针对用户问题 |
| Faithfulness | 忠实度 | 回答是否基于证据 |
| Contextual Relevancy | 上下文相关性 | 检索上下文是否相关 |
| Tool Correctness | 工具正确性 | 是否选对工具 |
| Hallucination Rate | 幻觉率 | 是否编造证据外信息 |

## 面试讲法

> Agent 项目不能只测最终文本，因为同一个问题可能有多种正确表达。我会先测确定性链路，比如路由、工具选择和工具返回结构；再用 DeepEval 这类大模型评测框架测回答质量、忠实度和幻觉率。
